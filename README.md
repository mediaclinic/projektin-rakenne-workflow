# Luo projektin rakenne
Automator skripti projektin rakenteen luomiseksi Finderissa.

Kansiorakenne:

```
{ Kansion nimi }
 |
 +-- 01_Aineistot
		 +-- Kuvat
		 +-- Referenssit
		 +-- Tekstit
 |
 +-- 02_Projektitiedostot
		 +-- Arkisto
 |
 +-- 03_Exports
 |
 +--04_Vedokset
 |
 +--05_Valmiit
		+-- Lähetetty
```

## Huomioita
Projektitiedostojen sisällä oleva Arkisto on tarkoitettu aikaisemmille versioille, uusin käytössä oleva versio tulisi olla Projektitiedostokansion juuressa.

Kun tiedosto on lähetetty eteenpäin painoon, lehteen tai vastaavaan, siirrä tiedosto ”Lähetetty” kansioon, josta tiedetään tiedoston lähteneen suunnittelijalta.
